可从机场出来一直表现得十分礼貌绅士的谢镜却没回应他，而是看着喻清棠和锦晏离去的方向。

喻盛和慕榆都有些不解，正想叫谢镜的时候，谢镜自己却回了神。

他若无其事地说自己对环境的适应性很强，不会挑剔住处，随便什么地方都行。

但这话喻盛却是不信的。
